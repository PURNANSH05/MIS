# Quick Export Reference Guide

## Where to Export Each Report

### 1. Dashboard Summary Report
📍 **Location**: Dashboard (Main Menu)
🔘 **Button**: Top right - "Export [Export]" button
📥 **File Name**: `dashboard-summary-[timestamp].csv`
📊 **Contains**: KPIs, totals, key metrics
⏱️ **Export Time**: < 1 second

### 2. Audit Log Report
📍 **Location**: Audit Logs (Main Menu)
🔘 **Button**: Top right - "Export" button  
📥 **File Name**: `audit-report-[timestamp].csv`
📊 **Contains**: User activity, actions, changes, IP addresses
⏱️ **Export Time**: < 2 seconds
🔐 **Classification**: Confidential

### 3. Inventory Stock Report
📍 **Location**: Inventory (Main Menu)
🔘 **Button**: Top right - "Export [Export]" button
📥 **File Name**: `inventory-report-[timestamp].csv`
📊 **Contains**: All items, SKUs, quantities, reorder levels, expiry dates
⏱️ **Export Time**: < 2 seconds

### 4. Batch Details Report
📍 **Location**: Inventory → Click Item → Details
🔘 **Button**: "Download CSV" button (batch section)
📥 **File Name**: `batch-details-[SKU]-[timestamp].csv`
📊 **Contains**: Batch numbers, manufacturing dates, expiry dates
⏱️ **Export Time**: < 1 second

### 5. Stock Report
📍 **Location**: Reports → Stock Tab
🔘 **Button**: Top right - "Export Stock" button
📥 **File Name**: `stock-report-[timestamp].csv`
📊 **Contains**: Item, SKU, category, quantity, reorder level
⏱️ **Export Time**: < 2 seconds

### 6. Expiry Report ⚠️ CRITICAL
📍 **Location**: Reports → Expiry Tab
🔘 **Button**: Top right - "Export Expiry" button
📥 **File Name**: `expiry-report-[timestamp].csv`
📊 **Contains**: Items with expiration dates, batch info, location
⏱️ **Export Time**: < 2 seconds
🚨 **Priority**: HIGH - Review immediately

### 7. Movement Report
📍 **Location**: Reports → Movements Tab
🔘 **Button**: Top right - "Export Movements" button
📥 **File Name**: `movements-report-[timestamp].csv`
📊 **Contains**: All transactions, timestamps, quantities, references
⏱️ **Export Time**: < 2 seconds

## How to Open Exported Files

### Microsoft Excel
1. Open Excel
2. File → Open
3. Select your CSV file
4. Click Open

### Google Sheets
1. Go to sheets.google.com
2. File → Import
3. Upload tab → Select your CSV
4. Choose import settings
5. Create new spreadsheet

### LibreOffice Calc
1. Open LibreOffice Calc
2. File → Open
3. Select CSV file
4. Choose delimiter settings
5. Click OK

### Apple Numbers
1. Open Numbers
2. File → Open
3. Select your CSV file
4. Numbers converts automatically

## Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Export button not showing | Check permissions, refresh page |
| File is empty | Verify data exists, try again |
| Won't open in Excel | Use "Data" → "From Text" import wizard |
| Data looks corrupted | Check file encoding (UTF-8), try diff app |
| File won't download | Check browser downloads, try different browser |
