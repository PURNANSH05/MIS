## Troubleshooting

### "Export button not showing"
→ Check permissions, refresh browser

### "File is empty"
→ Verify data exists in system

### "Won't open in Excel"
→ Use "Data" → "From Text" import

### "Data looks strange"
→ Check file encoding (UTF-8)

### "Download failed"
→ Check internet, try different browser
